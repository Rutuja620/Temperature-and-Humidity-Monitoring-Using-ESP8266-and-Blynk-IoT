# Temperature and Humidity Monitoring Using ESP8266 and Blynk IoT

## Description
This project measures temperature and humidity using a DHT sensor with ESP8266 NodeMCU. The sensor readings are displayed on the Arduino IDE Serial Monitor and also monitored remotely through the Blynk IoT Cloud platform in real time using Wi-Fi connectivity.

## Components Used
- ESP8266 NodeMCU
- DHT11/DHT22 Sensor
- Jumper Wires
- Breadboard
- USB Cable

## Software Used
- Arduino IDE
- Blynk IoT Platform

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- Wireless data transmission using Wi-Fi
- Serial Monitor output
- Blynk cloud monitoring

## Working
1. ESP8266 reads temperature and humidity from DHT sensor.
2. Sensor values are displayed on Arduino IDE Serial Monitor.
3. Data is sent to Blynk IoT Cloud through Wi-Fi.
4. User can monitor values remotely using Blynk app/dashboard.

## Applications
- Smart Home Monitoring
- Weather Monitoring System
- IoT-based Environmental Monitoring

## Outcome
Successfully monitored temperature and humidity values locally and remotely using ESP8266 and Blynk IoT Cloud.
